Setters

Color
Sets the color that will be used when filling and stroking shapes. A byte ranges from 0 to 255. 

void setColor(byte r, byte g, byte b);

Alpha
Sets the transparency of the filling and stroking color. The argument 'a' is a decimal number from 0 to 1. 

void setAlpha(float a);

Stroke width
Sets the stroke width used when stroking shapes. The argument 'w' is greater than 0. 

void setStrokeWidth(float w);

Window size
Changes the size of the window. 

void setWindowSize(int w, int h);

Font size
Changes the font size. 

void setFontSize(int s);
