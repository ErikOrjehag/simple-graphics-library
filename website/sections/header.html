Simple Graphics Library

SGL is a small graphics wrapper for GDI+ designed to help windows C++ developers to draw stuff with very little effort.

The setup process is simple. There are no external dependences and no need to link to libraries. Please feel free to improve the source via GitHub.

The sample code to the right is all you need to open a window and draw a moving rectangle!

Download SGL Contribute on GitHub 

#include "SGL.h"
#include <cmath>

void main() {

  int t = 0;

  sgl->setColor(129, 98, 179);

  while(true)
  {
    sgl->clear(255, 255, 255);

    sgl->fillRect(
      200 + 100 * sin(0.05 * t),  // pos x
      200 + 100 * cos(0.05 * t),  // pos y
      50, 50                      // size
    );

    sgl->render();

    t++;

    Sleep(10);
  }
}
