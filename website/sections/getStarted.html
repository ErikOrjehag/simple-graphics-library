How to get started

Setting up SGL is easy and you can do it in under 5 minutes. When your download is complete, extract the files and place them in your project.

Make sure that your SubSystem is set to WINDOWS . If you don't do this you'll probably get an error looking something like this. That's because the WinMain functions never gets called and therefore SGL is never initialized. Here is an image showing you have to set the correct SubSystem in Visual Studio.

Create a new cpp file and copy paste the sample code from the right box into your project.

Congratulations, you just made your first SGL program! Smack that compile button and enjoy the show. 

#include "SGL.h"
#include <cmath>

void main() {

  int t = 0;

  while(true)
  {
    t++;

    sgl->clear(255, 255, 255);

    sgl->setColor(
      (255/2) + (255/2) * cos(0.005 * t),
      (255/2) + (255/2) * sin(0.005 * t),
      (255/2) + (255/2) * cos(0.01 * t)
    );

    float xpos = 600 + 400 * sin(0.01 * t);
    float ypos = 300 + 100 * cos(0.05 * t);

    sgl->save();
    sgl->translate(xpos, ypos);
    sgl->rotate(t);
    sgl->fillRect(-25, -25, 50, 50);
    sgl->restore();

    sgl->render();

    Sleep(10);
  }
}
