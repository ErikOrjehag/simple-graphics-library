Drawing

Render
When you are done with your fills and strokes, call the render method to reveal your masterpience to the screen. 

void render();

Clear
Clear the entire screen by filling it with a rgb color. A byte ranges from 0 to 255. 

void clear(byte r, byte g, byte b);

Filling
Fills a shape with the current seleced color. The arguments are pretty self explanatory. 

void fillRect(float x, float y, float w, float h);
void fillTriangle(float x, float y, float w, float h);
void fillCircle(float x, float y, float r);
void fillEllipse(float x, float y, float w, float h);
void fillPolygon(Gdiplus::Point &points, int count);

Stroking
Strokes a shape with the current selected color and stroke width. The arguments are pretty self explanatory. 

void strokeRect(float x, float y, float w, float h);
void strokeTriangle(float x, float y, float w, float h);
void strokeCircle(float x, float y, float r);
void strokeEllipse(float x, float y, float w, float h);
void strokePolygon(Gdiplus::Point &points, int count);

Polygon
Here is an example of how to fill a polyon. The polygon can be ether convex or concave, both work just as well. The same principle applies when stroking polygons. 

using namespace Gdiplus

Point points[] = {
  Point(0, 0),
  Point(0, 50),
  Point(50, 50),
  Point(50, 0),
  Point(25, 30)
};

sgl->fillPolygon(*points, 5);

Line
Draw a line from start to end with the current selected color and stroke width. 

void drawLine(float startx, float starty, float endx, float endy);

Path
Use moveTo(x, y) to set the pen position. Use lineTo(x, y) to draw a line from that position to a new position. Continue the path by using multiple lineTo(x, y) calls. 

void moveTo(float x, float y);
void lineTo(float x, float y);

Text
Draws a string at the specified position. Look at the example for usage. 

void drawString(const WCHAR str[], float x, float y);

// Example
sgl->drawString(L"Hello world!", 40, 20);

Image
Draws an image at the specified position. Look at the example for usage. 

void drawImage(Gdiplus::Image *img, float x, float y);

// Example
using namespace Gdiplus
Image img(L"example.png");
sgl->drawImage(&img, 100, 100);
